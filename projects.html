Projects

This page highlights three projects I have completed through my coursework. These projects reflect my experience with programming, engineering design, data analysis, and computer systems.

Arduino RC Car

Course: MME 1005 Introduction to Engineering Design

In this project, my team designed and built an RC car that had to complete a track as quickly as possible. The car was controlled through a phone, which required both physical design work and software setup.

My Role:

  • Helped design the body of the RC car.
  • Worked on code that allowed the board to connect to a phone.
  • Contributed to the overall design process and testing.

Skills Used: Arduino, engineering design, problem solving, teamwork, coding

What I Learned:

This project gave me hands-on experience combining hardware and software in one design. I learned more about prototyping, testing, and how design choices affect overall performance.

Sorting Project

Course: CSC-2710 Advanced Data Structures / Algorithm Development

This project compared multiple sorting algorithms using datasets of 1,000 and 10,000 elements. We analyzed how different algorithms performed depending on dataset size, order, and type.

My Role:

  • Wrote the code for Heap Sort and Selection Sort.
  • Helped graph the results from the sorting tests.
  • Helped analyze comparisons, swaps, and runtime performance.

Skills Used: C++, algorithms, data analysis, graphing, runtime comparison

What I Learned:

This project helped me better understand algorithm efficiency beyond just learning Big-O notation in class. It showed how data arrangement and dataset size can change the actual performance of an algorithm in practice.

Trivia Game Server

Course: CSC-2410 Computer Systems

In this project, we created a trivia game in C++ that connected to a Raspberry Pi server. We also studied the performance differences between single-threaded and multithreaded approaches.

My Role:

  • Helped develop the C++ trivia game system.
  • Worked with the Raspberry Pi server connection setup.
  • Participated in analyzing single-thread versus multithread performance.

Skills Used: C++, Raspberry Pi, computer systems, multithreading, performance analysis

What I Learned:

This project gave me experience working with systems-level programming and server communication. It also helped me understand how threading can improve performance depending on the type of task being completed.
