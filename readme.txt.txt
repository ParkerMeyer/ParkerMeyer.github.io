Parker Meyer Portfolio Website

This is my personal portfolio website made for my Webdesign class. the purpose of this site is to showcase my resume, taken courses, projects, and info about me.

This is a very simple 6 page site with a style sheet in order to add some detail to the site. I used Bootstap, and W3 schools to figure out tons of things with this site like the nav bar, and a functional footer. I also used AI to help with adding comments, and formatting.

Overall I do like the site layout alot and I like the design, but I feel I could've done much more with the style sheet to add more depth to the site.
